Hello fellow developer!

Below you find some useful customization for your workflow:


Branch name scheme:

Simple flow: {key}-{formated name} / Short flow: {key} / GitLab flow: {GitLab flow issue type}/{key}-{formated name}


Copy to clipboard comand: git checkout -b "branch name"

Examle: \[.*?\] with flag g - Removing all brackets with content inside

Branch key registry:

Lowercase: ars-1-{name}
Uppercase: {key}-BRANCH-NAME, Lowercase: {key}-branch-name

* If at the time of installing the plugin, you have open tabs with Jira, you need to refresh them.
