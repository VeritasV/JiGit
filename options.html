Hello fellow developer!

Below you find some useful customizations for JiGit plugin that converts Jira issue title
to branch name in the format you need and puts it to clipboard.

Branch name scheme:
Simple flow: {issueKey}-{formated name}, Short flow: {issueKey}, ID flow: {issueId}
GitLab flow: {GitLab flow issue type}/{issueKey}-{formated name}
Copy with command
Copy to clipboard comand: git checkout -b "branch name"
Examle: \[.*?\] with flag g - Removing all brackets with content inside
Branch key registry:
Lowercase: ars-1-{name}
Uppercase: {key}-BRANCH-NAME, Lowercase: {key}-branch-name
Feedback / Issue
* If at the time of installing the plugin, you have open tabs with Jira, you need to refresh them.
